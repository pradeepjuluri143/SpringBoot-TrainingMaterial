üîÑ While Loop

While loops execute code repeatedly while a condition is true.

public class WhileDemo { public static void main(String[] args) { int count = 1; while(count <= 5) { System.out.println("Count: " + count); count++; } // Sum calculation int sum = 0, num = 1; while(num <= 5) { sum += num; num++; } System.out.println("Sum: " + sum); } }

üîÅ Do-While Loop

Do-while loops execute at least once, then check condition.

public class DoWhileDemo { public static void main(String[] args) { int num = 1; do { System.out.println("Number: " + num); num++; } while(num <= 3); // Runs at least once even if condition is false int x = 10; do { System.out.println("x = " + x + " (runs once)"); } while(x < 5); } }
Number: 1
Number: 2
Number: 3
x = 10 (runs once)

üîß Methods

Methods are reusable blocks of code that perform specific tasks.

public class MethodDemo { public static int add(int a, int b) { return a + b; } public static void greet(String name) { System.out.println("Hello, " + name + "!"); } public static int findMax(int[] arr) { int max = arr[0]; for(int num : arr) { if(num > max) max = num; } return max; } public static void main(String[] args) { greet("Java"); int sum = add(15, 25); System.out.println("Sum: " + sum); int[] nums = {45, 12, 78, 23}; System.out.println("Max: " + findMax(nums)); } }
Hello, Java!
Sum: 40
Max: 78

üìä Arrays

Arrays store multiple values of the same type.

public class ArrayDemo { public static void main(String[] args) { // Array declaration and initialization int[] numbers = {10, 20, 30, 40}; String[] languages = new String[3]; languages[0] = "Java"; languages[1] = "Python"; languages[2] = "C++"; System.out.println("First number: " + numbers[0]); System.out.println("Array length: " + numbers.length); System.out.println("All numbers:"); for(int i = 0; i < numbers.length; i++) { System.out.println("Index " + i + ": " + numbers[i]); } System.out.println("Languages:"); for(String lang : languages) { System.out.println("- " + lang); } } }
